# Pine Script for Zed

Basic local Zed extension for Pine Script files.

## Features

- Associates `.pine` and `.pinescript` files with Pine Script.
- Adds simple Tree-sitter based syntax highlighting.
- Adds common Pine Script snippets.
- Supports line comment toggling with `//`.

## Install Locally

1. Open Zed.
2. Run `zed: install dev extension` from the command palette.
3. Select this folder:


This extension is intentionally small. It is useful for editing Pine Script locally, but it does not provide TradingView-level compile diagnostics.
